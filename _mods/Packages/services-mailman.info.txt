configuration help:
You may want to add that line into your apache conf if you use rewrite rules
RewriteRule  ^/int([0-9]+)  /tiki-integrator.php?repID=$1  [QSA,L]
For this feature works, the apache user has to be granted the permission to 
exectue the list_lists mailman command. You can use visudo and add
www-data ALL= NOPASSWD: /usr/sbin/list_lists
